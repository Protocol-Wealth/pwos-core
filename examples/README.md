# Examples

Private workspace examples for composing PWOS Core packages. Examples are
hermetic by default and must not require live credentials for build/test.

## Available

- `rias-agent-substrate/` — reference implementation of the three-tier agent
  memory architecture, composing `@protocolwealthos/audit-log` and
  `@protocolwealthos/ai-guardrails`.

Future example backlog is tracked in
[#74](https://github.com/Protocol-Wealth/pwos-core/issues/74) rather than
maintained as a stale unchecked list here.

Contributions welcome - see [CONTRIBUTING.md](../CONTRIBUTING.md).
